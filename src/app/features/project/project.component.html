Show Time Calculator

Ce projet consiste à la réalisation d'une application web qui permet aux utilisateurs de calculer le temps nécessaire dont ils ont besoin afin de compléter une série ou un anime. Le calcul peut se faire sur une série ou un anime que l'utilisateur veut commencer ou qu'il a déjà commencé. Le calcul s'effectue également par rapport au nombre d'épisodes ou le nombre d'heures que l'utilisateur peut regarder par jour.

La stack
Angular 14, Bootstrap 5.2, Firebase
